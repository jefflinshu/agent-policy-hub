# Capafy Policy Hub

This folder is intended to become a GitHub Pages-backed policy hub for Capafy Agents.

Suggested future URL pattern:

- `/resume-generator/privacy-policy`
- `/release-calendar/privacy-policy`
- `/infocard/privacy-policy`
- `/agent-listing-studio/privacy-policy`

Recommended hosting approach:

1. Put this folder in a public GitHub repository.
2. Enable GitHub Pages for the repository.
3. Keep one markdown page per Agent policy.
4. Reuse the same structure for future terms, support, or changelog pages.
